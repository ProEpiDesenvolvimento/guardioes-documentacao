# Sobre o Projeto

O **Guardiões da Saúde** nasceu para unir tecnologia, ciência de dados e participação cidadã na vigilância epidemiológica.

Seu objetivo é permitir que cidadãos comuniquem sintomas e sinais de doenças em tempo real, ajudando na detecção precoce de surtos.

O sistema é composto por:
- Aplicativo móvel (coleta de sintomas)
- Painel web (visualização e análise)
- API pública (integrações)
